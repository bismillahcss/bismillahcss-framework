BismillahCSS Documentation

Utility-first CSS framework. No JS dependencies. Mobile-first responsive components. Fully customizable with CSS variables.

1. Installation

npm

npm install bismillahcss

CDN

<link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bismillahcss/dist/bismillah.min.css" />

Manual

Copy `src/bismillah.css` and the `src` folder into your project, or import from package build output.

2. Get Started

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8" />
  <meta name="viewport" content="width=device-width, initial-scale=1.0" />
  <link rel="stylesheet" href="https://cdn.jsdelivr.net/npm/bismillahcss/dist/bismillah.min.css" />
</head>
<body>
  <div class="bismillah-container">
    <h1 class="bismillah-text-3xl bismillah-font-bold">Hello BismillahCSS</h1>
    <button class="bismillah-btn bismillah-btn-primary">Primary</button>
  </div>
</body>
</html>

3. Core Theme Variables

Customize your design system by overriding CSS variables before loading BismillahCSS.

:root {
  --bismillah-primary-color: #1e40af;
  --bismillah-secondary-color: #64748b;
  --bismillah-success-color: #16a34a;
  --bismillah-warning-color: #eab308;
  --bismillah-danger-color: #dc2626;
  --bismillah-info-color: #0ea5e9;
  --bismillah-bg-color: #f8f9fa;
  --bismillah-text-color: #212529;
  --bismillah-border-radius: 8px;
}

8. Next.js Integration

Use BismillahCSS in a Next.js project via global CSS import.

App Router (Next.js 13+)

// app/layout.js
import 'bismillahcss/dist/bismillah.min.css';
import './globals.css';

export default function RootLayout({ children }) {
  return (
    
      {children}
    
  );
}

// pages/_app.js
import 'bismillahcss/dist/bismillah.min.css';
